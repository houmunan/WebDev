总结

常用Selectors，三个

Element: 
	li {

}
class: 
	.hello
id: 
	#name {

}

其他重要Selectors

* (star selector，通配符，选择页面全部) 
	* {

}
Descendent Selectors, 两个及以上的Selectors, 洋葱式的大的包含小的 
	li a {
	选择所有li里面的a的东西
}
	ul li a {
	选择所有在ul里面的li的里面的a的东西
}
	li .hello{
	选择所有在li里面的class是.hello的东西
}
Adjacent Selctor, 选择比如说所有h1后头跟着的h4或者li之类的，same level不是下级level 
	h4 + ul{
	选择所有h4后面的ul的东西
}
Attribute Selector, based on ANY attributes, like href, img, type, etc. 
	a[href="http://www.google.com"] {
	选择所有链接指向google.com的元素，施加比如background: blue;
}
	input[type="checkbox"] {
	background: blue;
}
nth of number, 选择每个element的第n个 
括号里3代表整页中第三个表，或者even代表所有第偶数个表:
	ul:nth-of-type(3){
	background: purple;
}
括号里3代表所有表中的第三行，或者even代表所有表中的偶数行：
	li:nth-of-type(3){
	background: purple;
}

CSS Font Stack:

    font-family: "Arial"; /*如果字体不是阿拉伯数字开头，则不加“双引号”也可以*/
}

front-size

既可用font-size连px决定字体大小，也可用font-size连em决定字体对于原本大小的倍数大小

(段落里的两个词用<span></span>括起来了，作用和div相同，单纯选定这两个词。)
#span1 {
    font-size: 50px; /*字体变为50px*/
}
#span2 {
    font-size: 2.0em /*字体变为原本2.0倍:一般设置body用px定基础大小，再用em定其他所有h1,h2,p,span,div等。*/
}

使用Google Fonts (其他也行)：

<link rel="stylesheet" type="text/css" href="http://fonts.googleapis.com/css?family=Raleway:400,700|Indie+Flower">
在css文件里使用字体：直接使用即可。
